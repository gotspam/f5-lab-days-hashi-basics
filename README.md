## Overview

These lab exercises are intended to provide hands on experience with on-boarding F5 BIG-IP and automating common tasks using Terraform.

## Prerequisites 

1. Access to the F5 Unified Demo Framework (UDF)
2. Chrome browser 

## Environment

The UDF blueprint consists of the following resources:

- 2 x BIG-IP virtual appliances (15.0.x)
- 1 Ubuntu 18 application server running: 
    - NGINX
- 1 Ubuntu 18 client server running: 
    - Coder (Web based VS Code) 
    - Docker 
        - f5-cli container
    - Sphinx
