F5 Automation With Terraform
============================

This class covers the following topics:

- How to use Terraform and DO to automate BIG-IP infrastructure onboarding
- How to use Terraform and AS3 to automate BIG-IP configuration changes
- Incorporate security automation to eliminate friction and security/infrastructure bottlenecks

Expected time to complete: **1.5 hours**

.. note::

   All work for this lab will be performed exclusively from the Client
   Server. No installation or interaction with your local system is
   required.
